# Specification

## Summary
**Goal:** Build a simple case converter tool that transforms text between uppercase and lowercase.

**Planned changes:**
- Create a centered multi-line text input area with placeholder text
- Add an UPPERCASE button that converts all text to uppercase
- Add a lowercase button that converts all text to lowercase
- Apply a clean, modern, centered design with minimal aesthetic

**User-visible outcome:** Users can type or paste text into a text area and instantly convert it to uppercase or lowercase by clicking the respective buttons.
